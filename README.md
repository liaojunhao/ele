# vuejs仿写饿了么APP

在学习vue的时候写的一个小demo，通过多写项目提高对vue框架的深入了解

## 主要技术栈：
* vuejs2.0
* vue-router
* vue-resource
* vue-cli ( 脚手架 )
* webpack ( 打包工具 )
* es6+eslint
* 1像素边框实现技巧

## 开发环境：
* windows
* 开发方式：工程化、组件化、模块化
